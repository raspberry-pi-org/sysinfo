## OS
* `Raspberry Pi 4B 4GB`

## Env
* `Python 3.7`
* `apscheduler`
* `baidu-aip`
* `Baidu AIP SDK( aip-python-sdk-2.2.15 )`

## How To Run On RasPi 4B
you need to install the dependencies firstly
> $ pip3 install apscheduler

> $ cd aip-python-sdk-2.2.15

> $ pip3 install baidu-aip

> $ python3.7 setup.py install

then run it
> $ python3.7 system.py
